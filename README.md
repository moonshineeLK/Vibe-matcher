# 🌷 vibe-matcher

A fun Python program that matches your aesthetic based on your hobbies.

## How it works
- Enter your name and hobby
- Choose which category your hobby fits into
- Get your personal aesthetic + a vibe description

## Aesthetics
1. Athletic → Sportcore 💪🏻
2. Creative → Artcore 🌸
3. Intellectual → Dark Academia 🍁
4. Tech → Cyberpunk 👩🏻‍💻
5. Self-loving → Cottagecore 🌼
6. Chill → Dreamcore ✨🌙
7. Explorer → Nomadic Naturalist 🌾

## How to run
Run vibe_matcher.py in any Python environment.
